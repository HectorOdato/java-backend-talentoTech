# 📚 Sistema de Gestión de Librería en Java

Proyecto integrador desarrollado en Java para la gestión de stock y venta de libros.

## 🚀 Funcionalidades

- Agregar libros
- Listar libros
- Buscar libros por ID
- Actualizar precio y stock
- Eliminar libros
- Crear pedidos
- Validación de stock
- Manejo de excepciones personalizadas

## 🛠️ Tecnologías utilizadas

- Java
- Programación Orientada a Objetos (POO)
- ArrayList
- Manejo de excepciones
- Git y GitHub

## 📂 Estructura del proyecto

src/
├── main/
├── model/
├── service/
├── exception/

## 📚 Conceptos aplicados

- Encapsulamiento
- Clases y objetos
- Colecciones dinámicas
- Separación de responsabilidades
- Validaciones
- Excepciones personalizadas
